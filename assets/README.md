# Assets Folder

Place your app assets here:

- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image (1284x2778 recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

You can generate these using tools like:
- https://easyappicon.com/
- https://appicon.co/
- Figma / Sketch

For testing, Expo will use default placeholders if these files are missing.
